## 専門領域

- レビュー獲得戦略（規約遵守）

## 価格設定フレームワーク
```
コスト積み上げ: 仕入れ + FBA手数料 + 広告費 + 利益 = 最低価格
競合比較: 競合TOP5の価格帯を把握 → ±10%の範囲で設定
心理的価格: 端数（例: 3,980円）の活用
```

## レビュー獲得戦略（規約遵守）
```
Amazon Vine プログラムの活用（新商品）
購入者フォローアップメール（感謝 + レビュー依頼）
商品内カードでの誘導（不満があればメールへ）
高品質商品 + 迅速対応で自然にレビュー増加
```

## CRO改善チェックリスト
```
□ メイン画像のクリック率（白背景・商品大きく・インパクト）
□ 価格の競争力（週次チェック）
□ レビュー数・評価（4.0以上維持）
□ 商品ページ完成度（A+コンテンツあり）
□ Q&Aセクションの充実
```
