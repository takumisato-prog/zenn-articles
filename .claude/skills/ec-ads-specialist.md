## Amazon商品ページ最適化

### タイトル設計（A10アルゴリズム対応）
```
構造: [ブランド名] [メインキーワード] [サイズ/容量/仕様] [ターゲット] [特徴]
文字数: 150〜200文字（カテゴリによる上限あり）
必須要素:
  - 月間検索ボリューム1,000以上のキーワードを前半に配置
  - 競合差別化ポイントを含める
  - 禁止事項: 「No.1」「最安値」「保証」等の誇大表現
```

### 箇条書き5点（ベネフィット→フィーチャー順）
```
各項目の構造:
  【ベネフィット（感情訴求）】 詳細説明（機能・スペック）
  例: 【快眠を実現する高密度フォーム】 反発弾性率25%の高密度ウレタン採用。体圧を均等分散し...

配置戦略:
  1点目: 最重要ベネフィット（購入決定理由No.1）
  2点目: 差別化ポイント（競合との違い）
  3点目: 使用シーン・ターゲット訴求
  4点目: 品質・安全性・認証
  5点目: アフターサポート・保証
```

### バックエンドキーワード設定
```
- 250バイト以内（半角スペース区切り）
- タイトル・箇条書きと重複NG
- ロングテールキーワード・表記揺れ・英語名を含める
- Subject Matter: 商品カテゴリを正確に設定
```

### A+コンテンツ設計
```
モジュール構成（推奨）:
  1. ブランドストーリー（ヘッダー画像 + キャッチコピー）
  2. 商品特徴（テキスト + 画像 × 3点）
  3. 比較表（自社ラインナップ or 競合比較）
  4. 使用シーン・ライフスタイル画像
  5. FAQ・補足情報

プレミアムA+（資格取得後）:
  - 動画埋め込み・インタラクティブモジュール活用
```

## Amazon SEO戦略（A10アルゴリズム）

### A10アルゴリズムの主要ランキング要因
| 要因 | 重要度 | 対策 |
|------|--------|------|
| 販売速度（Sales Velocity） | ★★★★★ | 広告・クーポンで初速UP |
| クリック率（CTR） | ★★★★☆ | メイン画像・価格・レビュー最適化 |
| 転換率（CVR） | ★★★★☆ | 商品ページ品質・価格競争力 |
| レビュー評価・件数 | ★★★★☆ | Vine・購入者フォローアップ活用 |
| キーワード関連性 | ★★★☆☆ | タイトル・箇条書き・ST最適化 |
| セラー実績（FBA率等） | ★★★☆☆ | FBA利用・カート獲得率維持 |

### キーワード調査プロセス
```
1. 競合上位10商品のタイトル・箇条書きからキーワード抽出
2. Helium10 Cerebro / Jungle Scout Keyword Scout で検索ボリューム確認
3. Amazon Suggest（オートコンプリート）で関連ワード収集
4. 分類: コアキーワード（月1万↑）/ サブキーワード（月1千〜）/ ロングテール（月百〜）
5. 配置優先度: タイトル > 箇条書き1-2行目 > 説明文 > バックエンド
```

## Amazon広告（AmazonAds）運用

### キャンペーン設計（フルファネル）

```
【認知】スポンサーブランド（SB）
  - ブランドキーワードターゲティング
  - 競合ブランドターゲティング
  - 動画広告（SBV）: CTR 2倍効果

【検討】スポンサーディスプレイ（SD）
  - 商品ターゲティング（競合ASIN）
  - オーディエンス: 商品カテゴリ閲覧者

【購買】スポンサープロダクト（SP）
  - オートキャンペーン（新規キーワード発掘）
  - マニュアル: 完全一致（コンバージョン重視）
  - マニュアル: フレーズ一致（ボリューム確保）
```

### KPI管理基準
| KPI | 目標値（カテゴリ平均） | アクション |
|-----|---------------------|----------|
| ACOS | 15〜25% | 20%超でビッド下げ・キーワード除外 |
| ROAS | 4〜7倍 | 3倍未満でキャンペーン構成見直し |
| CTR | 0.3%以上 | 0.1%未満でクリエイティブ変更 |
| CVR | 10%以上 | 10%未満で商品ページ改善 |
| Impression Share | 上位20%以内 | 競合に負けていたらビッドUP |

### 週次最適化チェックリスト
```
□ 検索用語レポート確認 → 高ACOS除外キーワード追加
□ オートキャンペーンの良好ワード → マニュアルに昇格
□ 入札額調整（ACoS目標比較）
□ 予算消化率確認（日予算×消化率）
□ 競合価格チェック・価格調整の要否判断
```

## Google広告運用

### キャンペーン構成
```
検索広告:
  - ブランドキーワードキャンペーン（ROAS重視）
  - 商品カテゴリキャンペーン（コンバージョン重視）
  - 競合指名キャンペーン（CVR計測重視）

ショッピング広告:
  - 優先度設定: P-MAX > スマートショッピング廃止後
  - フィード最適化: タイトル・説明文・GTINの精度

P-MAXキャンペーン:
  - アセット充実（テキスト15本・画像20枚・動画5本目標）
  - オーディエンスシグナル設定必須
  - コンバージョン計測精度の確認
```

### 品質スコア改善
```
品質スコア = 広告関連性 × LP品質 × 予想CTR
7点以上を目標（10点満点）

対策:
- SKAG（Single Keyword Ad Group）でキーワード↔広告文の一致度UP
- LPのファーストビューにキーワードを含める
- 広告表示オプション（サイトリンク・コールアウト）の充実
```
