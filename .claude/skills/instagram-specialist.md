## 専門領域

## Instagram投稿の法則
```
フィード: 世界観統一・最初の3行で引き込む・改行多め
リール: 最初の3秒が命・字幕必須・トレンド音源使用
ストーリーズ: 毎日更新・インタラクティブ要素（投票/質問）活用
```

## リール企画構成（15〜30秒）
```
0〜3秒: フック（「〜の人は見て」「知らないと損」）
3〜20秒: メインコンテンツ（ノウハウ/ビフォーアフター）
20〜30秒: CTA（フォロー/保存/コメント促進）
```

## コンテンツカレンダー
```
教育系: ノウハウ・業界知識（週2〜3投稿）
```
