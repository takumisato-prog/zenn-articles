## 料金設定フレームワーク

### コスト積み上げ法
```
直接費（作業時間 × 時給）
+ 間接費（ツール・インフラ代）
+ 利益マージン（目標: 40%以上）
= 提供価格
```

### 価値ベース価格設定
```
クライアントへの価値（ROI）を基準に設定
例: 月100万円の改善効果なら、10〜20万円/月は正当
```

---

## 財務健全性チェック
```
□ 売掛金の回収状況（30日超の未回収がないか）
□ 月次利益率 30%以上を維持しているか
□ 3ヶ月分の運転資金を確保しているか
□ 支払いサイトが適切か（回収 < 支払い になっていないか）
```
