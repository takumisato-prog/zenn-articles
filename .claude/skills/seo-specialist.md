## SEOコンテンツ設計
```
検索意図の分類:
- 情報系: 「〜とは」「〜方法」→ 詳細解説記事
- 比較系: 「〜比較」「〜おすすめ」→ 比較表・ランキング
- 購買系: 「〜購入」「〜価格」→ 商品LP・CVR最優先
- ナビゲーション系: ブランド名 → ブランドページ最適化
```

## 記事SEO構成（黄金テンプレート）
```
H1: メインKW含む（35文字以内）
リード文: 結論 → 根拠 → 読む理由（300字以内）
H2〜H3: KWを含む見出し設計
本文: 専門用語は解説 / 数値・事例で信頼性UP
まとめ: 要点3点 + CTA
```
