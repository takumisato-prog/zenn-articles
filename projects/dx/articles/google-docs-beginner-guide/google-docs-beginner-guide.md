# Googleドキュメントの使い方【IT未経験者向け完全ガイド2026年版】

「Wordと何が違うの？」「共同編集って怖くない？」「誰かに上書きされたらどうしよう…」——Googleドキュメントを初めて使う方から、こんな不安をよく聞きます。

Googleドキュメントは、**インターネット上で文書を作成・編集・共有できる無料ツール**です。Wordに似ていますが、複数人が同時に編集できる・自動保存される・どのデバイスからでも開けるという点で、チームでの業務に特に適しています。

この記事では、IT未経験の方に向けて、Googleドキュメントの基本操作から共同編集・コメント・バージョン履歴まで、ひとつひとつ丁寧に解説します。

---

## GoogleドキュメントとWordの違い——まずここを理解しよう

Googleドキュメントを使い始める前に、MicrosoftのWordとの違いを理解しておきましょう。

[IMAGE: 01-docs-vs-word.svg]
Alt text: "GoogleドキュメントとWordの違い比較図"

| 比較項目 | Microsoft Word | Googleドキュメント |
|---------|--------------|-----------------|
| 料金 | 有料（Microsoft 365が必要） | 無料（Googleアカウントがあれば使える） |
| 保存方法 | 手動で「上書き保存」が必要 | **自動保存**（保存操作不要） |
| 保存場所 | パソコン本体・OneDrive | **Google Drive（クラウド上）** |
| 別の端末からのアクセス | 設定が必要 | **どこからでも即開ける** |
| 複数人での同時編集 | 難しい | **リアルタイムで同時編集可能** |
| バージョン管理 | 手動でファイルを複製 | **自動で変更履歴を保存** |
| オフライン編集 | 可能 | 設定が必要（制限あり） |

**最大のポイントは「自動保存」と「同時編集」です。** 保存し忘れてデータが消えた、というミスがなくなり、複数人で1つの文書を同時に編集できます。

> **よくある誤解：** 「Wordで作ったファイルはGoogleドキュメントで開けないのでは？」——開けます。Word形式（.docx）のファイルをGoogleドライブにアップロードすると、Googleドキュメントとして編集できます。

## STEP 1：Googleドキュメントを作成する

### 新しい文書を作る方法

**方法①：Googleドライブから作成**
1. ブラウザで `drive.google.com` にアクセスしてログイン
2. 「＋新規」ボタンをクリック
3. 「Googleドキュメント」を選択
4. 新しい文書が開く

**方法②：直接アクセスして作成**
- ブラウザのアドレスバーに `docs.new` と入力してEnterを押すだけで、新しいGoogleドキュメントが即座に開きます。

### 文書のタイトルをつける

新しい文書を開くと、上部に「無題のドキュメント」と表示されます。ここをクリックしてタイトルを入力しましょう。

> **ポイント：** タイトルは後から探しやすい名前にしましょう。「議事録_20260319_定例会議」のように日付や内容を入れると管理しやすくなります。

## STEP 2：基本操作——テキストの入力と書式設定

### テキストの入力

文書エリアをクリックしてカーソルを置いたら、そのままキーボードで文字を入力できます。**自動保存されるので、入力した内容はリアルタイムでGoogleドライブに保存されます。**

### 主な書式設定

| 操作 | ショートカット | 効果 |
|------|------------|------|
| 太字 | Ctrl+B（Mac: ⌘+B） | **太字**にする |
| 斜体 | Ctrl+I（Mac: ⌘+I） | *斜体*にする |
| 下線 | Ctrl+U（Mac: ⌘+U） | 下線を引く |
| コピー | Ctrl+C（Mac: ⌘+C） | 選択箇所をコピー |
| 貼り付け | Ctrl+V（Mac: ⌘+V） | 貼り付け |
| 元に戻す | Ctrl+Z（Mac: ⌘+Z） | 直前の操作を取り消す |

### 見出しを使う

ツールバーの「標準テキスト」というドロップダウンを「見出し1」「見出し2」に変更すると、文書に階層構造を持たせられます。**見出しをつけることで、自動的に目次が生成でき、長い文書でも読みやすくなります。**

## STEP 3：共同編集——複数人で同時に作業する

Googleドキュメントの最大の特徴が**共同編集**です。複数の人が同じ文書を同時に開いて編集でき、お互いの変更がリアルタイムで画面に反映されます。

### 共有する手順

1. 文書を開いた状態で、右上の「**共有**」ボタンをクリック
2. 共有相手のメールアドレスを入力
3. 権限を選択する

### 3つの権限モードを理解しよう

[IMAGE: 02-collab-modes.svg]
Alt text: "Googleドキュメント 編集・提案・閲覧モードの違い"

権限には3種類あります。

| 権限 | 何ができる？ | いつ使う？ |
|------|-----------|----------|
| **編集者** | 自由に編集・削除できる | 一緒に作業するチームメンバー |
| **コメント可** | コメントを追加できる（本文は変えられない） | レビューや確認をお願いするとき |
| **閲覧者** | 見るだけ（変更不可） | 完成した文書を共有するとき |

> **初心者へのアドバイス：** 「誤って内容を書き換えられたくない」場合は「コメント可」が最適です。相手は意見やフィードバックをコメントで残せますが、本文は変更できません。

### 編集モードと「提案モード」

文書を開いたとき、画面右上に「**編集**」と表示されている場合は自由に編集できます。これを「**提案**」に切り替えると、変更箇所が色付きで表示され、文書のオーナーが承認・却下できる形になります。

**提案モードの使い方：**
1. 画面右上の「編集」ドロップダウンをクリック
2. 「提案」を選択
3. 文字を編集すると、変更箇所が緑色で表示される
4. 文書のオーナーが「✓（承認）」か「✗（却下）」を選択

これにより、**「誰がどこを変更したか」が一目でわかります。**

## STEP 4：コメント機能——フィードバックをわかりやすく残す

コメント機能を使うと、文書内の特定の箇所にメモや質問を残せます。メールで「〇ページの〇行目の部分ですが…」と長々説明する必要がなくなります。

### コメントの追加方法

1. コメントをつけたいテキストを選択（ドラッグで範囲指定）
2. 画面右側に表示される「コメントを追加」アイコン（吹き出しマーク）をクリック
3. コメントを入力して「コメント」ボタンをクリック

**または：** テキストを選択後に `Ctrl+Alt+M`（Mac: `⌘+Option+M`）でもコメントを追加できます。

### コメントで特定の人にメンションする

コメント入力欄で `@メールアドレス` または `@名前` と入力すると、その人に通知が届きます。「この部分、〇〇さんに確認してほしい」というときに便利です。

### コメントを解決する

対応が完了したコメントは「**解決**」ボタンをクリックすることで、画面から非表示になります（削除ではなく、履歴として残ります）。

> **活用例：** 上司が「ここの表現を変えてください」とコメント → 担当者が修正 → 「解決」をクリックして完了報告。メールのやり取りなしで、フィードバックから修正確認まで完結します。

## STEP 5：バージョン履歴——過去の状態に戻す

「誰かが文書を書き換えてしまった」「昨日の時点の内容に戻したい」——そんなときに役立つのがバージョン履歴です。Googleドキュメントは**変更のたびに自動でバージョンを保存**しています。

[IMAGE: 03-version-history.svg]
Alt text: "Googleドキュメント バージョン履歴の使い方フロー図"

### バージョン履歴の確認方法

1. メニューバーの「**ファイル**」をクリック
2. 「**バージョン履歴**」→「**バージョン履歴を表示**」をクリック
3. 画面右側に日時ごとの変更履歴が表示される
4. 確認したいバージョンをクリックすると、その時点の文書を見られる

### 以前のバージョンに戻す方法

1. バージョン履歴を開く
2. 戻したいバージョンを選択
3. 「**このバージョンを復元**」ボタンをクリック

> **安心ポイント：** 復元しても、それ以降の変更履歴はすべて残ります。「間違って復元してしまった」場合も、再度バージョン履歴から元に戻せます。

### バージョンに名前をつける

重要な節目（例：「社長確認済み」「最終版」）には、バージョンに名前をつけておくと、後から探しやすくなります。

1. バージョン履歴を開く
2. 名前をつけたいバージョンの右にある「︙」をクリック
3. 「このバージョンに名前をつける」を選択

## よくある質問（FAQ）

### Googleドキュメントは無料ですか？

Googleアカウントがあれば無料で使えます。会社でGoogle Workspaceを契約している場合は、より高度な機能（大容量ストレージ・管理者機能など）が利用できます。

### Wordファイルに変換してダウンロードできますか？

できます。「ファイル」→「ダウンロード」→「Microsoft Word (.docx)」を選択するとWordファイルとしてダウンロードできます。PDF形式でもダウンロード可能です。

### インターネットがない環境でも使えますか？

ChromeブラウザでGoogleドキュメントのオフラインアクセスを設定しておくと、インターネットなしでも編集できます。ただし、その間は他の人との同期はできません。

### 文書が突然消えてしまった場合は？

バージョン履歴から復元できます。また、誰かが削除した場合はGoogleドライブのゴミ箱（30日間保持）から復元できます。

### コメントの通知メールがたくさん届いて困る

Googleドキュメントの通知設定から、「コメントの通知」を変更できます。「メンションされたときのみ」に設定すると、関係するコメントのみ通知が届くようになります。

## まとめ：Googleドキュメントを使いこなす5つのポイント

1. **保存は自動**——Ctrl+Sを押す必要はない。変更は即座に保存される
2. **共有権限は用途に応じて使い分ける**——閲覧・コメント可・編集の3種類を意識する
3. **フィードバックはコメント機能で**——メールよりも速く、正確に伝えられる
4. **変更は提案モードで**——誰が何を変えたか明確になり、レビューがスムーズ
5. **バージョン履歴は万能の保険**——失敗を恐れず使える安心感の源

Googleドキュメントは使えば使うほど、チームの文書作成が効率化します。まずは自分ひとりで文書を作ることから始めて、慣れてきたら共同編集・コメント機能を活用してみてください。
