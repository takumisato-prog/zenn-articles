## 1. ポジショニング

**ターゲット読者**
- Claude Code を使い始めた個人・フリーランス・小規模事業主
- AI活用で業務効率化したい人（エンジニア以外も含む）
- 副業・起業を考えているエンジニア

**差別化ポイント**
- 「Claude Codeで会社を作った」という体験談ベース
- コピペで使えるCLAUDE.md / スキルファイルの実物公開
- 5部門エージェント連携という具体的なシステム設計

---

## 2. プラットフォーム戦略

| プラットフォーム | 役割 | 課金形態 |
|---------------|------|---------|
| **note** | メインの収益基盤 | 単記事 500〜1,500円 / マガジン 2,980円 |
| **Zenn** | エンジニア層への認知獲得 | 無料記事（note誘導） |
| **X（Twitter）** | 集客・認知拡大 | 無料（フォロワー獲得） |

**基本フロー**
```
Zenn（無料・技術寄り）→ X拡散 → note（有料・実践寄り）
```

---

## 3. コンテンツ設計

### フェーズ1：認知獲得（0〜1ヶ月）
無料記事3本でZenn・Xに投稿し、フォロワーと読者を獲得する。

### フェーズ2：収益化（1〜3ヶ月）
有料記事・有料マガジンをnoteで販売開始。

### フェーズ3：テンプレート販売（3ヶ月〜）
実際のCLAUDE.md・スキルファイルをセット商品として販売。
