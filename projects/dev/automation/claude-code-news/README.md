# Claude Code ニュース収集スクリプト

Claude Codeの最新情報を自動収集し、Googleスプレッドシートにまとめる。

## 収集対象

| ソース | 内容 |
|--------|------|
| GitHub claude-code リリース | バージョンアップ・新機能 |
| Anthropic公式ブログ | 公式発表・研究 |
| Zenn Claude Codeトピック | 日本語Tips・使い方 |
| Zenn Claudeトピック | Claude全般の日本語記事 |
| Reddit r/ClaudeAI | 英語コミュニティTips |

## セットアップ手順

### 1. スプレッドシートを用意する

**新規作成する場合：**
1. `script.google.com` で新規プロジェクト作成
2. スクリプトを貼り付けて保存
3. `createNewSpreadsheet()` を実行
4. ログに表示されたIDをコピー

**既存シートを使う場合：**
- スプレッドシートのURLから `/d/XXXX/` の `XXXX` 部分をコピー

### 2. スクリプトプロパティを設定する

「プロジェクトの設定」→「スクリプトプロパティ」に追加：

| プロパティ名 | 値 |
|-------------|-----|
| `SHEET_ID` | スプレッドシートのID |
| `NOTIFY_EMAIL` | 通知先メールアドレス（不要なら空欄） |

### 3. 初回実行（権限承認）

「実行」→ `collectClaudeCodeNews` を選択して実行。
権限承認ダイアログが出るので承認する。

### 4. トリガー設定

「トリガー」→「トリガーを追加」：
- 実行する関数: `collectClaudeCodeNews`
- イベントのソース: 時間主導型
- 時間ベースのトリガーのタイプ: 日付ベースのタイマー
- 時刻: 午前7時〜8時

## スプレッドシートの構成

**新着ログシート（自動作成）：**

| 取得日時 | ソース名 | タイトル | 概要 | URL |
|---------|---------|---------|------|-----|
| 2026/03/25 07:00 | GitHub claude-code リリース | v1.x.x リリース | ... | リンク |

## ソースの追加・削除

スクリプト冒頭の `RSS_SOURCES` 配列を編集：

```javascript
const RSS_SOURCES = [
  {
    name: "表示名",
    url: "RSSフィードのURL",
    enabled: true,  // false にすると無効化
  },
  // ...
];
```

## コスト

- **完全無料**（Claude API不使用）
- GASの無料枠内で動作
