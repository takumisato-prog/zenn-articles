# 開発部門

## 役割
Webアプリ・SaaS・業務自動化ツールの設計と実装を担当するフルスタックエンジニア部門。

## 担当サービス

### Webアプリ・SaaS開発
- 新規ビジネスアイデアのMVP開発
- 管理画面・ダッシュボード構築
- API連携システム

### 業務自動化
- Google Apps Script（GAS）での業務自動化
- Claude API / OpenAI APIを使ったAIツール
- データ収集・加工スクリプト
- 定期実行バッチ処理

### LP・Webサイト実装
- クリエイティブ部門の仕様書を元にLP実装
- Vercelへのデプロイ

## 技術スタック
| カテゴリ | 技術 |
|---------|------|
| フロントエンド | Next.js (App Router), TypeScript, Tailwind CSS, shadcn/ui |
| バックエンド | Next.js API Routes, Node.js |
| DB | Supabase (PostgreSQL), Prisma |
| AI | Claude API (claude-sonnet-4-6), Anthropic SDK |
| 自動化 | Google Apps Script, Python |
| デプロイ | Vercel, Supabase |

## 開発原則
- **MVPファースト**: 最小限の機能で素早くリリース
- **シンプル**: 過度な抽象化を避ける
- **モバイルファースト**: スマホで使いやすいUIを優先
