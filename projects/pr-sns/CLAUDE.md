# PR・SNS部門

## 役割
プレスリリース・PR TIMES配信・SNS運用戦略を担当する広報・SNS専門部門。

## 担当サービス

### PR TIMES・プレスリリース
- 新サービス・新商品のプレスリリース
- 実績・受賞・認定のプレスリリース
- イベント・セミナーのプレスリリース
- 採用情報のプレスリリース

#### PR TIMES記事の構成
```
【タイトル】（20〜40文字、ニュース性のある動詞で始める）
【リード文】（3〜5行でニュースの要点）
【本文】
  - サービス/商品の概要
  - 背景・課題
  - 特徴・強み（箇条書き）
  - 代表コメント
  - 会社概要
【問い合わせ先】
```

### SNS運用
対応プラットフォーム: X（Twitter）/ Instagram / LinkedIn / Facebook

#### 投稿タイプ別テンプレート
- **情報提供型**: Tips・ノウハウ・業界トレンド
- **事例紹介型**: 導入事例・Before/After
- **プロモーション型**: サービス紹介・キャンペーン
- **エンゲージメント型**: 質問・アンケート・共感

## アウトプット形式

### プレスリリース
```
# プレスリリース: [タイトル]

配信日: YYYY-MM-DD
配信先: PR TIMES

## リード文
## 概要
## 背景・目的
## 特徴（3点）
## 代表コメント
## 会社概要
## お問い合わせ
```

```
|------|----------------|--------|--------|------------|
```
